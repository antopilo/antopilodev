NUAKE GAME ENGINE

===========

A game engine inspired by Quake. Made to integrate with Trenchbroom and uses Wren as a scripting language. It features a deferred PBR rendering system with Jolt Physics.

PIXEL SORTER STUDIO

===========

A standalone image processing software made to Pixel Sort images and export them. Essentially a port of an existing After Effects Plugin.


Website

NUAKE RT

===========

A real-time raytracing software with basic scene system that uses Compute Shaders and temporal accumulation. Implemtation based on Raytracing In One Weekend with my own improvements along the way.


Github

LIBRARIES

NUAKE RENDERER

===========

A simple rendering library made initially for Nuake Engine. A library I used in NuakeUI, NuakeRT and most of my toy projects that requires rendering.


Github

NUAKE UI

===========

A UI library using Nuake Renderer. Made to be fast and flexible, it uses Yoga Layout as a layout engine. The interface is parsed from XML and CSS and the behaviour is defined in C++.
